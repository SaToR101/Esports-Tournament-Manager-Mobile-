# Esports Tournament Manager (Mobile)

Это мобильное приложение на React Native (Expo), созданное для управления киберспортивными турнирами.

## Установка и запуск

1.  **Установите зависимости:**
    ```bash
    npm install
    ```

2.  **Запустите проект:**
    ```bash
    npx expo start
    ```

3.  **Запуск на телефоне:**
    *   Установите приложение **Expo Go** (из App Store или Google Play).
    *   Отсканируйте QR-код из терминала.

## Структура проекта

*   `App.tsx`: Точка входа в приложение.
*   `src/screens`: Экраны приложения (Главная, Детали, Настройки).
*   `src/components`: Переиспользуемые компоненты.
*   `src/context`: Контексты (Тема, Язык).
*   `src/lib`: Утилиты (Хранилище, Переводы).

## Технологии

*   **React Native / Expo**: Фреймворк.
*   **NativeWind**: Стилизация (Tailwind CSS для React Native).
*   **React Navigation**: Навигация.
*   **AsyncStorage**: Локальное хранение данных.
*   **Lucide React Native**: Иконки.
